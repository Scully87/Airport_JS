How to use
------------
Clone the repository:
```shell
$ git clone git@github.com:scully87/airpot_js.git
```

Change into the directory:
```shell
$ cd airport_js
```

Open browser to see tests:
```shell
$ open Specrunner.html
```

Play when in browser:
```shell
 command + alt + I
```
